About

Created by Joran Schols. A personal project focused on astronomy and finding the right conditions to watch the stars.

What this is

This project started as a way to quickly understand the night sky — light conditions, weather and essential astronomical data — without noise or clutter.

What it does

Forecast

A clear overview of upcoming weather and star-watching conditions.

Sun & Moon

Sunrise, sunset and daily moon phases.

Planets

Rise and Set of all planets in our solar system.

Location-based

Tailored to any location you choose.

APOD

Watch the Nasa Astronomy Picture of the Day.

Alerts

Coming Soon: Receive notifications when ideal stargazing conditions occur.

How it’s built

Built with Angular, .NET and public APIs.
Designed to be fast, readable and easy to maintain.

APIs:
Open-Meteo
NASA APOD API
OpenStreetMap

Libraries:
SunCalc
Astronomy Engine
